Dashboard del vehículo implementado en una pantalla **Nextion**.

## Descripción

Este repositorio contiene los recursos necesarios para el funcionamiento del dashboard del vehículo en una pantalla Nextion. Incluye el diseño de la interfaz HMI, el archivo compilado para cargar en la pantalla y la documentación de uso.

## Contenido del repositorio

* **FSULL DashBoard.HMI**
  Proyecto editable del dashboard para el editor de Nextion.

* **FSULL DashBoard.tft**
  Archivo compilado listo para cargar en la pantalla Nextion.

* **Manual Dashboard CRT - v2024-25.pdf**
  Manual del sistema y funcionamiento del dashboard.

* **Recursos/**
  Carpeta con recursos gráficos o archivos auxiliares utilizados por el proyecto.

## Uso

1. Abrir el archivo `.HMI` en el **Nextion Editor** para modificar la interfaz.
2. Compilar el proyecto para generar el archivo `.tft`.
3. Cargar el `.tft` en la pantalla Nextion mediante tarjeta microSD o conexión directa.
